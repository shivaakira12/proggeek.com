2.Variables and Datatypes

Variables :

A Variable is the name given to a memory location in a program. A variable is container which is used to hold the variable.Variables and data types in python as the name suggests are the values that vary. In a programming language, a variable is a memory location where you store a value. The value that you have stored may change in the future according to the specifications.

Rules for Declaring the variable:

1. A variable name can't start with digit.
2. No whitespaces allowed to be used inside a variable name.
3. A variable name can start with an alphabet and underscore.
4. A variable name can contain alphabets,digits and underscores.

Operators:

Operators are used to make operations such as addition,subtraction,multplication and division.

1.Assignment Operators(=,+=,-= etc)
2.Comparison Operators(=>,<=<,>,++,--)
3.Logical Operators(and,or,not)
4.Assignment Operators(+,-,*,/)

Datatypes:

Numerical data type holds numerical value. In numerical data there are 4 sub types as well. Following are the sub-types of numerical data type:

1.Integers
2.Float
3.Complex Numbers
4.Boolean

Integers are used to represent whole number values.

Float data type is used to represent decimal point values.

Complex numbers are used to represent imaginary values. Imaginary values are denoted with ‘j’ at the end of the number.

Boolean is used for categorical output, since the output of boolean is either true or false.
