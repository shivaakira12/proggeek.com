What are the main Differences between Compiler and Interpreter ?

Compiler


  • Compiler Takes Entire program as input.
  • Intermediate object code is generated.
  • Conditional Control Statements executes faster.
  • It requires more memory
  • Program need not be compiled every time.
  • Errors are displayed after the entire program is checked
  • Example : C Compiler

    • Interpreter

  • Interpreter takes single instruction as input.
  • No Intermediate object code is generated.
  • Conditional Control Statements executes slower.
  • It requires less memory
  • Every time High level program is converted into low level program
  • Errors are displayed every instruction interpreted(if any)
  • Example : Python Interpreter
